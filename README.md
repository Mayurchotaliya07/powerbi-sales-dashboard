# 📊 Power BI Sales Dashboard

## 📌 Project Overview

This project presents an interactive Sales Dashboard built in **Power BI** using the **Sample Superstore** dataset. The dashboard helps analyze key business metrics, identify sales trends, monitor profitability, and compare performance across regions, categories, and customer segments.

---

## 🎯 Objective

The goal of this project is to create an interactive business intelligence dashboard that enables users to:

- Monitor overall sales performance
- Track profit and revenue trends
- Compare product categories and customer segments
- Analyze state-wise sales performance
- Support data-driven business decisions

---

## 📂 Dataset

- Sample Superstore Dataset
- Contains information about orders, products, customers, sales, profit, quantity, and regions.

---

## 🛠 Tools & Technologies

- Power BI
- Power Query
- DAX
- Data Modeling

---

## 📈 Dashboard KPIs

- Total Sales
- Total Profit
- Total Orders
- Total Quantity
- Average Order Value (AOV)

---

## 📊 Dashboard Visualizations

- Sales by State
- Sales by Category
- Sales by Segment
- Profit Trend over Time
- Profit by Sub-Category
- Sales vs Profit vs Quantity Analysis

---

## ✨ Features

- Interactive slicers
- Dynamic KPI cards
- Drill-down analysis
- Responsive visualizations
- Business performance tracking

---

## 💡 Key Insights

- Identify high-performing regions.
- Compare category-wise sales contribution.
- Analyze profitability across product sub-categories.
- Understand customer segment performance.
- Monitor sales and profit trends over time.
